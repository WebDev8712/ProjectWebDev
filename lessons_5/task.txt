a. Для чего используется flexbox?
a. Flexbox используется для создания гибких макетов на веб-странице, которые могут легко адаптироваться к различным размерам экранов и управлять распределением и выравниванием элементов внутри контейнера.

b. Как можно выравнивать элементы flexbox по горизонтали и вертикали?
b. Элементы flexbox могут быть выравнены по горизонтали с помощью свойства justify-content и по вертикали с помощью свойства align-items.

c. Как можно переносить элементы flexbox, если они не влезают в одну строку?
c. Если элементы flexbox не влезают в одну строку, то они автоматически переносятся на следующую строку. Это называется "автоматическим переносом строк" и управляется свойством flex-wrap.

d. Как изменять размеры элементов flexbox относительно друг друга?
d. Размеры элементов flexbox могут быть изменены с помощью свойства flex-grow для увеличения размера элемента и flex-shrink для уменьшения размера элемента относительно других элементов.

e. Для чего используется grid?
e. Grid используется для создания более сложных и точных макетов на веб-странице с помощью сетки из строк и столбцов.

f. Как устанавливаются размеры сетки? Какие единицы измерений при этом можно использовать?
f. Размеры сетки устанавливаются с помощью свойств grid-template-rows и grid-template-columns, где вы указываете размеры строк и столбцов. Можно использовать различные единицы измерения, такие как px, %, fr (доля доступного пространства).

g. Как размещаются по ячейкам сетки её элементы?
g. Элементы сетки размещаются по ячейкам сетки с помощью свойства grid-column и grid-row, которые определяют начальную и конечную точку для каждого элемента.

h. Как можно выравнивать элементы внутри ячейки? по простому и коротко
h. Элементы внутри ячейки сетки могут быть выравнены с помощью свойства justify-self для выравнивания по горизонтали и align-self для выравнивания по вертикали.
