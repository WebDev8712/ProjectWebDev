a. Как можно подключить JS-код в HTML-документ?
a. JS-код можно подключить в HTML через тег `<script>` или внешний файл с помощью атрибута `src`.

b. Как описываются переменные в JS?
b. Переменные в JS описываются с помощью `var`, `let` или `const`, за которым следует имя переменной и (необязательно) значение.

c. Как можно получить доступ к элементу HTML из JS?
c. Для доступа к элементу HTML из JS используется метод `document.getElementById()`.

d. Как можно из JS менять текст HTML-элемента?
d. Изменить текст HTML-элемента можно с помощью свойств `innerHTML` или `innerText`.

e. Как можно менять из JS атрибуты HTML-элементов?
e. Атрибуты HTML-элементов изменяются с помощью метода `setAttribute()`.

f. Как можно добавлять из JS HTML-элементы?
f. Чтобы добавить HTML-элементы, используйте методы `createElement()` для создания нового элемента и `appendChild()` для его добавления в DOM.

g. Как получить текстовое значение HTML-элемента?
g. Текстовое значение HTML-элемента можно получить с помощью свойств `textContent` или `innerText`.

h. Как получить HTML-код элемента?
h. HTML-код элемента можно получить через свойство `innerHTML`.
