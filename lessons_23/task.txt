a. Какой синтаксис условного оператора в JS?
a. Условный оператор в JS имеет следующий синтаксис:
javascript
if (условие) {
    // код, выполняемый, если условие истинно
} else if (другое_условие) {
    // код, выполняемый, если другое_условие истинно
} else {
    // код, выполняемый, если ни одно из условий не истинно
}

b. Что является логическими значениями в JS?
b. Логическими значениями в JS являются `true` (истина) и `false` (ложь).

c. Как работает тернарное условие в JS?
c. Тернарное условие в JS имеет следующий формат:
javascript
условие ? выражение1 : выражение2;

Если условие истинно, возвращается выражение1, иначе - выражение2.

d. Какие циклы есть в JS?
d. В JS есть три типа циклов:
   - Цикл `for`: для выполнения кода определенное количество раз.
   - Цикл `while`: для выполнения кода до тех пор, пока условие истинно.
   - Цикл `do...while`: для выполнения кода хотя бы один раз, а затем повторения, пока условие истинно.

e. Как в JS получать доступ к элементам внутри других элементов?
e. Для доступа к элементам внутри других элементов в JS используется метод `querySelector()` или `querySelectorAll()`, указывая селектор, который идентифицирует нужный элемент.

Пример:
javascript
var innerElement = document.querySelector('#outerElement #innerElement');

Этот код найдет элемент с id "outerElement", а затем внутри него найдет элемент с id "innerElement".
