a. Что в JS делает ключевое слово async?
a. Ключевое слово `async` в JavaScript используется для обозначения, что функция является асинхронной. Это позволяет использовать ключевое слово `await` внутри этой функции.

b. Что в JS делает ключевое слово await?
b. Ключевое слово `await` в JavaScript используется в асинхронных функциях для приостановки выполнения кода до тех пор, пока не завершится выполнение асинхронной операции, после чего она возвращает результат.

c. Что в JS делает функция fetch()?
c. Функция `fetch()` в JavaScript используется для отправки сетевых запросов и получения данных из сети. Она возвращает объект Promise, который позволяет обрабатывать результаты запроса асинхронно.
