
a. Как работают переходы в CSS?
a. Переходы в CSS создают плавные изменения свойств элементов при изменении их состояния.

b. Как работают анимации в CSS?
b. Анимации в CSS позволяют создавать динамичные изменения свойств элементов во времени.

c. Что такое ключевые кадры?
c. Ключевые кадры (keyframes) в CSS определяют основные состояния анимации.

d. Что такое временные функции?
d. Временные функции в CSS управляют изменением скорости анимации с течением времени.
