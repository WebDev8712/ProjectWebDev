a. Какими тегами можно добавить списки в HTML-документ?
a. Списки в HTML можно создать с помощью тегов:
   - Неупорядоченный список: \<ul>
   - Упорядоченный список: \<ol>
   - Элемент списка: \<li>

b. Какие атрибуты могут быть у списков?
b. У списков могут быть атрибуты, такие как type для типа маркера и start для начального числа.

c. Как добавить картинку в HTML-документ?
c. Чтобы добавить картинку, используйте тег \<img> с атрибутом src для пути к изображению.

d. Какие атрибуты могут быть у картинок?
d. Для изображений могут использоваться атрибуты src для пути к изображению, alt для альтернативного текста и другие для управления размером и поведением.

e. Какое основное правило вложенных тегов?
e. Основное правило вложенных тегов: вложенные теги должны быть правильно расположены внутри друг друга.

f. В чём отличие одиночных и парных тегов?
f. Одиночные теги не имеют закрывающего тега и используются для создания элементов без содержимого, в то время как парные теги имеют начальный и закрывающий теги и обрамляют содержимое.
